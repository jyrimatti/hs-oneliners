Haskell one-liners - lahteenmaki.net

One-liners to launch a complete Haskell development environment.

No dependencies needed except for Nix.

REPL

  • Launch GHCi REPL.
  • Give Haskell packages as PKGS environment variable.
  • If the only parameter is a cabal file, the repl contains Haskell/system packages defined in the cabal file.
						
                                         sh <(curl https://hs.lahteenmaki.net/repl)
PKGS='lens aeson'                        sh <(curl https://hs.lahteenmaki.net/repl)
PKGS=myHaskellProjects/hello/hello.cabal sh <(curl https://hs.lahteenmaki.net/repl)

Shell

  • Launch a shell with GHC and Haskell development tools.
  • In the shell you can start your favorite editor.
  • Give Haskell packages as PKGS environment variable.
  • If the only parameter is a cabal file, the shell contains Haskell development tools and GHC with Haskell/system packages defined in the cabal file.
						
                                         sh <(curl https://hs.lahteenmaki.net/shell)
PKGS='lens aeson'                        sh <(curl https://hs.lahteenmaki.net/shell)
PKGS=myHaskellProjects/hello/hello.cabal sh <(curl https://hs.lahteenmaki.net/shell)

Editor

  • Launch your default editor with GHC and Haskell development tools.
  • Give Haskell packages as PKGS environment variable.
  • If the only parameter is a cabal file, your default editor is opened in the parent folder with Haskell development tools and GHC with Haskell/system packages defined in the cabal file.
						
                                         sh <(curl https://hs.lahteenmaki.net/editor)
PKGS='lens aeson'                        sh <(curl https://hs.lahteenmaki.net/editor)
PKGS=myHaskellProjects/hello/hello.cabal sh <(curl https://hs.lahteenmaki.net/editor)

Sublime Text

  • Launch Sublime Text (if installed) with GHC and Haskell development tools.
  • Give Haskell packages as PKGS environment variable.
  • If the only parameter is a cabal file, Sublime is opened in the parent folder with Haskell development tools and GHC with Haskell/system packages defined in the cabal file.
						
                                         sh <(curl https://hs.lahteenmaki.net/sublime)
PKGS='lens aeson'                        sh <(curl https://hs.lahteenmaki.net/sublime)
PKGS=myHaskellProjects/hello/hello.cabal sh <(curl https://hs.lahteenmaki.net/sublime)
